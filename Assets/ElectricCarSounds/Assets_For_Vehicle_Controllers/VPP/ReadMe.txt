If you want to use Electric Car Sounds with Vehicle Physics Pro,
import Vehicle Physics Pro, Electric Car Sounds and VPP_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Vehicle Physics Pro. This prefabs are ready to use with Vehicle Physics Pro.
Get Vehicle Physics Pro here: https://vehiclephysics.com
or download the free Vehicle Physics Pro- Community Edition here: https://assetstore.unity.com/packages/tools/physics/vehicle-physics-pro-community-edition-153556?aid=1101l3r5B

Read the documentation for tutorials.
