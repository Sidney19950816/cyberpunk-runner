If you want to use Electric Car Sounds with Driver (Traffic System),
import Driver (Traffic System), Electric Car Sounds and later Driver_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Driver (Traffic System). This prefabs are ready to use with Driver (Traffic System).
If you don't own Driver (Traffic System), you can get it here: https://assetstore.unity.com/packages/templates/driver-traffic-system-69703?aid=1101l3r5B

Read the documentation for tutorials.
