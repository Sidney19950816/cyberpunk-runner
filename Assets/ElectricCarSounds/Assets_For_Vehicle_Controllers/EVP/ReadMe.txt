If you want to use Electric Car Sounds with Edy's Vehicle Physics,
import Edy's Vehicle Physics, Electric Car Sounds and later EVP_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Edy's Vehicle Physics. These prefabs are ready to use with Edy's Vehicle Physics 5.
If you don't own Edy's Vehicle Physics, you can buy it here: https://assetstore.unity.com/packages/tools/edy-s-vehicle-physics-403?aid=1101l3r5B

Read the documentation for tutorials.
