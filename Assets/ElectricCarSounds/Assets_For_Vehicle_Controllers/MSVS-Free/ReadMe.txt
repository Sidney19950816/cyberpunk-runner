If you want to use Electric Car Sounds with MS Vehicle System - Free,
import MS Vehicle System - Free, Electric Car Sounds and later MSVS-Free_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for MS Vehicle System. This prefabs are ready to use with MS Vehicle System vehicles.
MS Vehicle System - Free is available here for free: https://assetstore.unity.com/packages/tools/physics/ms-vehicle-system-free-version-90214?aid=1101l3r5B

Read the documentation for tutorials.
