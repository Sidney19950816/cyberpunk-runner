If you want to use Electric Car Sounds with RCK(Realistic Car Kit),
import Realistic Car Kit, Electric Car Sounds and later RCK_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Realistic Car Kit. This prefabs are ready to use with Realistic Car Kit vehicles.
Get Realistic Car Kit here: https://assetstore.unity.com/packages/tools/physics/realistic-car-kit-18421?aid=1101l3r5B

Read the documentation for tutorials.
