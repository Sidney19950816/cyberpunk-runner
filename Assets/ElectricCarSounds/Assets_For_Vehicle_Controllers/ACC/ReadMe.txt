If you want to use Electric Car Sounds with Arcade Car Controller,
import Arcade Car Controller (make sure you done all the required steps to make ACC work), then import Electric Car Sounds and later ACC_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and a prefab for Arcade Car Controller.
If you don't own Arcade Car Controller, you can buy it here: https://assetstore.unity.com/packages/templates/systems/arcade-car-controller-142660?aid=1101l3r5B
Read the documentation for tutorials.
