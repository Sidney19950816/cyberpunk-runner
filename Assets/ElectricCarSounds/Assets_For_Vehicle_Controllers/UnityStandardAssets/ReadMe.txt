If you want to use Electric Car Sounds with Unity Standard Assets Car,
import Unity Standard Assets, Electric Car Sounds and later UnityStandardAssets_ECS.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Unity Standard Assets Car. This prefabs are ready to use with Unity Standard Assets Car.
You can get Unity Standard Assets here: https://assetstore.unity.com/packages/essentials/asset-packs/standard-assets-for-unity-2018-4-32351?aid=1101l3r5B

Read the documentation for tutorials.
